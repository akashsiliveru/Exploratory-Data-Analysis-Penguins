# About Penguin Species

There are **17 penguin species** on Earth, each with unique characteristics and habitats.  
In this project we focus on **three major penguin species** found in Antarctica.

###  Adélie Penguins
Adélie penguins are easily recognizable by their classic **black-and-white tuxedo appearance**.  
They are one of the most common penguin species in Antarctica and are well adapted to extreme environmental conditions.

###  Gentoo Penguins
Gentoo penguins have **bright red-orange bills and distinctive white head patches**.  
They are known for their fast swimming ability and relatively larger body size.

###  Chinstrap Penguins
Chinstrap penguins are named after the **thin black band under their head**, resembling a helmet strap.  
They are an important part of the Antarctic ecosystem.

# Dataset

The dataset used in this project contains **7 features describing penguin characteristics**.

| Column | Description |
|------|-------------|
| species | Penguin species (Adelie, Gentoo, Chinstrap) |
| island | Island where penguin was observed (Dream, Torgersen, Biscoe) |
| culmen_length_mm | Length of penguin bill in millimeters |
| culmen_depth_mm | Depth of penguin bill in millimeters |
| flipper_length_mm | Flipper length in millimeters |
| body_mass_g | Penguin body mass in grams |
| sex | Gender of penguin |

These features allow us to analyze **penguin morphology and distribution patterns**.

# Penguin Distribution Data Analysis Summary

Initial exploratory analysis revealed several interesting patterns:

- All **Chinstrap penguins** are located on **Dream Island**.
- All **Gentoo penguins** inhabit **Biscoe Island**.
- **Adélie penguins** are distributed across **all three islands**.
- **Torgersen Island** contains only **Adélie penguins**.
- Gender does **not significantly influence species classification**.
- Gender distribution across islands is **fairly balanced**.
- Because gender does not strongly influence classification, the **sex column could potentially be dropped in ML models**.
